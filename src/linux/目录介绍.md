# Linux 核心目录功能速查表
::: info /bin（基本工具箱）
存放 ls、cp 等常用命令，所有用户均可执行。
:::

::: important /boot（启动文件）
包含内核和引导程序，系统启动的核心文件目录。
:::

::: info /dev（设备接口）
将硬件设备以文件形式管理，如硬盘 /dev/sda、黑洞 /dev/null。
:::

::: tip /etc（系统配置）
存放系统和软件的核心配置文件，修改需 root 权限。
:::

::: important /home（用户目录）
每个普通用户的个人文件夹，用户对自身目录拥有完全权限。
:::

::: note /lib（系统库）
存放程序运行所需的共享库文件，类似 Windows 的 .dll 文件。
:::

::: warning /media（可移动设备）
U盘、光盘等可移动设备的自动挂载点。
:::

::: caution /tmp（临时文件）
短期存储临时数据，系统重启后自动清空，勿存重要内容。
:::

::: info /sys（硬件信息）
提供内核和硬件设备的实时数据，属于虚拟文件系统。
:::

::: warning /var（日志与缓存）
存放日志、缓存、邮件等动态变化的数据，/var/log 为核心日志目录。
:::

::: note /mnt（临时挂载）
手动挂载外部文件系统的专用目录，如挂载移动硬盘。
:::

::: info /opt（额外软件）
用于安装第三方额外软件包，软件独立存放便于管理。
:::

::: tip /proc（系统信息）
映射内存中的系统运行状态，/proc/cpuinfo、/proc/meminfo 可查看硬件信息。
:::

::: important /root（管理员目录）
超级用户 root 的专属家目录，普通用户无访问权限。
:::

::: caution /run（运行数据）
存放系统运行时的临时数据，重启后会清空。
:::

::: note /sbin（管理员命令）
存放 shutdown 等系统管理工具，仅 root 或 sudo 用户可执行。
:::

::: info /srv（服务数据）
存放服务器提供服务的相关数据，如网页、FTP 资源。
:::

::: tip /usr（软件资源）
存放系统大部分程序和库文件，是 Linux 最大的目录之一。
:::
